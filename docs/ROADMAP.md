# Implementation Roadmap

This document tracks the development phases for the AdistAppDemo.

## Phases
1.  **Phase 1: Authentication** - Setup Auth API, Repository, ViewModel, and Login UI.
2.  **Phase 2: Vehicle Discovery** - Fetch assigned vehicles and build the selection screen.
3.  **Phase 3: Vehicle Control** - Implement lock/unlock command logic and the control dashboard.
4.  **Phase 4: Refinement** - Branding, error handling, and demo-specific optimizations.
